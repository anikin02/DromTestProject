## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/anikin02/DromTestProject.git
    ```
2. Open the project in Xcode:
    ```bash
    cd DromTestProject
    open DromTestProject.xcodeproj
    ```
3. Run the project on a simulator or a real device.
